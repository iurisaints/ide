Letramento Digital Corporativo

Público-Alvo: Equipes administrativas, operacionais e de recepção.

Aumente a produtividade e a segurança de toda a sua empresa. Capacite sua equipe para usar as ferramentas de escritório (e-mail, planilhas, nuvem) de forma eficiente e segura.

Tópicos Principais
  • Boas Práticas de E-mail (Segurança, Phishing)
  • Organização de Arquivos na Nuvem (Google Drive / OneDrive)
  • Produtividade com Google Workspace / Microsoft 365
  • Segurança da Informação Básica (Senhas, 2FA)

Frontend para Não-Técnicos (HTML/CSS)

Público-Alvo: Times de Design, Marketing e Product Owners.

Faça seu time de design e marketing falar a mesma língua dos desenvolvedores. Dê autonomia para sua equipe fazer pequenas alterações em sites ou e-mail marketing sem depender 100% de TI.

Tópicos Principais
  • O que são HTML, CSS e JavaScript?
  • Estrutura básica de HTML (Tags principais)
  • Estilização básica com CSS (Cores, Fontes, Margens)
  • Como "Inspecionar Elemento" no navegador
  • Conceito de Responsividade (Mobile vs. Desktop)

Formação Backend Integral (APIs e Bancos)

Público-Alvo: Desenvolvedores Júnior ou equipes migrando de tecnologia.

Capacite sua equipe para construir a 'inteligência' por trás das aplicações. Da criação da API RESTful até a integração segura com o banco de dados.

Tópicos Principais
  • Fundamentos de API REST (Verbos HTTP, JSON)
  • Criação de Servidor (ex: Node.js e Express)
  • Modelagem de Bancos de Dados (SQL ou NoSQL)
  • Comandos CRUD (Create, Read, Update, Delete)
  • Autenticação e Segurança (JWT)

Formação Frontend Moderna

Público-Alvo: Desenvolvedores que precisam construir interfaces web modernas.

Modernize suas aplicações web. Treine sua equipe para construir interfaces rápidas, interativas e reutilizáveis com React.js, a tecnologia mais demandada do mercado.

Tópicos Principais
  • Nivelamento: JavaScript Moderno (ES6+)
  • Fundamentos do React: Componentes, Props e State
  • React Hooks (useState, useEffect)
  • Consumo de APIs (Integração com Backend)
  • Gerenciamento de Estado (Context API / Redux Toolkit)

Bootcamp Fullstack

Público-Alvo: Equipes de startups, programas de trainee ou times multifuncionais.

Formação completa de ponta a ponta. Sua equipe pronta para construir uma aplicação inteira, do clique do usuário até o salvamento no banco de dados, unindo React e Node.

Tópicos Principais
  • Fusão dos módulos de Backend e Frontend Moderno
  • Desenvolvimento de Projeto Prático Integrador
  • Boas Práticas de Integração (Frontend + Backend)
  • Conceitos de Deploy (Colocando a aplicação no ar)

Diagnóstico e Treinamento Cirúrgico

Público-Alvo: Equipes de desenvolvimento maduras com gargalos específicos.

Não compre um curso de prateleira. Compre a solução exata para o seu gargalo. Eu faço um diagnóstico da sua equipe e monto um treinamento 100% focado em resolver *o seu* problema.

Como Funciona
  • Fase 1: Diagnóstico (Entrevistas e análise de código)
  • Fase 2: Proposta Customizada (Plano de treinamento micro-focado)
  • Fase 3: Execução (O treinamento prático e focado)
